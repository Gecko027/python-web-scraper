# Python Web Scraper

This Python script scrapes structured data from a website and saves it into a CSV file.

## Features
- Extracts multiple items (quotes and authors)
- Parses HTML using BeautifulSoup
- Saves clean CSV output

## Dependencies
- Python 3.x
- requests
- beautifulsoup4

## How to Run
1. Clone the repo: `git clone <repo-url>`
2. Activate your virtual environment: `source venv/bin/activate`
3. Install dependencies: `pip install -r requirements.txt`
4. Run the scraper: `python scraper.py`
5. The data will be saved in `quotes.csv`
